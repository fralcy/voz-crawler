# VOZ Crawler

Dự án crawler cho diễn đàn VOZ - box "Tư vấn cấu hình".

## Mục đích
Crawl và phân tích dữ liệu từ các thread về tư vấn cấu hình máy tính trên VOZ.

## Cấu trúc dự án
- \`src/\`: Mã nguồn chính
- \`data/\`: Dữ liệu đã xử lý
- \`cache/\`: Cache tạm thời khi crawl
- \`docs/\`: Tài liệu dự án
